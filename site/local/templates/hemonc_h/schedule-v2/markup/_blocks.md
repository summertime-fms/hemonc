# Блоки для вёрстки (куда класть HTML)

Верстальщик работает в CSS и в `render()` / `mount()` соответствующих JS-модулей.
Ниже — чеклист блоков по ТЗ. Готовую вёрстку сюда не коммитим как отдельные HTML-фрагменты без нужды: источник правды — JS-шаблоны + CSS.

## 1. Фильтры — `schedule-v2-filters.js` + `.sch2-filters*`

- [ ] Специальность (мультивыбор / теги)
- [ ] Услуга или заболевание (автокомплит)
- [ ] Формат: Любой / Очно / Онлайн / На дому
- [ ] Кнопка «Ближайший приём»
- [ ] Мобильная шторка / гамбургер фильтров

## 2. Карточка врача — `schedule-v2-card.js` + `.sch2-card*`

- [ ] Фото, ФИО, специальность, стаж, степень
- [ ] Теги форматов: Очно / Онлайн / На дому
- [ ] Табы Первичный / Повторный
- [ ] Контейнер сетки слотов

## 3. Сетка слотов — `schedule-v2-slots.js` + `.sch2-slots*`

- [ ] Горизонтальный скролл по дням (сегодня + неделя)
- [ ] Слоты времени, hover/active
- [ ] Различие clinic / online / home
- [ ] Disabled (не подходит под visitType)
- [ ] Пустой день → «Показать ближайшие»

## 4. Запись — `schedule-v2-booking.js` + `.sch2-booking*`

- [ ] Сценарий А: контекст + ФИО + телефон + email + «Подтвердить запись»
- [ ] Сценарий Б: процедура на первом плане, врач вторичен

## 5. Состояния — `schedule-v2-list.js` + `.sch2-state*`

- [ ] Loading
- [ ] Error (+ повторить)
- [ ] Empty (+ сбросить фильтры / позвонить)

## Как смотреть

1. Открыть `/schedule-v2/` — песочница с моками.
2. Или на любой странице: `?schedule_v2=1` и клик «Запись на приём».
